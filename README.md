# Install Xray-core Mod
```html
mv /usr/local/bin/xray /usr/local/bin/xray.bakk && wget -q -O /usr/local/bin/xray "https://github.com/EZ-Code00/xray-core/releases/download/Xray-linux-64-v26.2.6-1/Xray-linux-64-v26.2.6-1" && chmod 755 /usr/local/bin/xray && systemctl restart xray && xray version
```
